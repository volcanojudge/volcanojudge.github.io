Solution to Knapsack Practice 1

Editorial by: Savir

This problem is a template 0-1 knapsack dynamic programming problem. The basic idea is to solve the problem by calculating answers to subproblems of the main problem.

In particular, for this problem, we can let $dp_{i_j}$ be the maximum sum of happiness values when considering the first $i$ items, and with a capacity of $j$ kg. Then, we can calculate $dp_{i_j}$ using subproblems as follows: $$dp_{i_j} = max(dp_{i_j}, dp_{i-1_j-w_i} + v_i)$$ (For all $1 \le i \le N$ and $1 \le j \le M$)

The time complexity and space complexity of the approach is $O(NM)$. It turns out we can reduce the space complexity to $O(M)$ by "reversing the loop". If we loop from $M$ to $w_i$ for each $i$, we can let $dp_j$ be the maximum sum of happiness values with a knapsack capacity of $j$ kg. $dp_j$ can now be calculated as follows: $$dp_j = max(dp_j, dp_{j-w_i} + v_i)$$ (For all $1 \le i \le N$ and $1 \le j \le M$)

Now, our final answer is $dp_M$.
